\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_mic_t_l_component_mag_compensator}{Mic\+T\+L\+Component\+Mag\+Compensator}} is a magnetic compensation algorithm based on a modified Tolles-\/\+Lawson model, which compensates the triaxial components of the magnetic-\/field vector. 

The corresponding compensation algorithm is originally proposed by Qinxuan Sun in \href{https://sunqinxuan.github.io/projects/2024-05-06-mag-compensation}{\texttt{ https\+://sunqinxuan.\+github.\+io/projects/2024-\/05-\/06-\/mag-\/compensation}}.
